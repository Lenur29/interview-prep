# Lemur

Full-stack web app for JavaScript interview preparation. Topic-based questions with self-rating and progress tracking.

## Status

🚧 Under active development.

## Tech stack

- **Frontend:** React 19, Vite, TypeScript, React Router v7, Apollo Client, Zustand, Base UI, Tailwind v4, Tiptap
- **Backend:** NestJS 11, Prisma, GraphQL (Apollo adapter), PostgreSQL
- **Auth + DB:** Supabase
- **Tooling:** pnpm workspaces, Turborepo, Biome, Vitest, Playwright, Lefthook
- **Hosting:** Vercel (web), Fly.io (api), Supabase (db + auth)

## Local development

Requires Node 24+ and pnpm 10+. Install pnpm via Homebrew (`brew install pnpm`) or corepack.

```sh
pnpm install
pnpm dev
```

More instructions added as the project grows.
